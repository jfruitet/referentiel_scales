Installation du module Référentiel

Le plugin Référentiel est composé de 3 parties :

- Un module (./mod/referentiel)
- Un bloc (./blocks/referentiel)
- Un rapport (./report/referentiel)

Téléchargez l'archive du module Référentiel correspondant à votre distribution de Moodle.
Il est composé de 3 archives :
referentiel-mod.zip à décomprimez dans le dossier ./moodle/mod/ de votre serveur.
referentiel-block.zip à décomprimez dans le dossier ./moodle/blocks/ de votre serveur
referentiel-report.zip à décomprimez dans le dossier ./moodle/report/ de votre serveur..

Activation du module Référentiel

Connexion à Moodle avec le compte "admin"
A la racine du site, cliquer sur la rubrique "Administration du site" / "Notifications" ; le nouveau module est installé...

Une fois l'installation achevée sans erreur (ça arrive aussi qu'il y ait des erreurs  ; dans ce cas signalez-les moi aussitôt avec une copie d'écran -- vous pouvez aussi passer en mode débug et m'indiquer le message d'erreur) cliquez sur
"Administration"/ "Administration du site" / "Plugins" / "Modules d'activités" / "Gestion des plugins"
et vous devriez voir la nouvelle activité "Référentiel" !

Vous pouvez alors la paramétrer au niveau du site. Ces paramètres permettent de limiter les capacités de modification / importation de référentiels par les utilisateurs et d'empêcher l'accès à des données de scolarité confidentielles ou non renseignées.

Activation du bloc Référentiel

A la racine du site, cliquer sur la rubrique "Administration du site" / "Notifications" ; le nouveau block est installé... cliquez sur ""Administration"/ "Administration du site" / "Plugins" / "Modules d'activités" / "Gestion des blocs"
et vous devriez voir le nouveau bloc "Référentiel"

Cliquez sur l'icône "oeil" pour rendre le bloc disponible dans les cours...

Activation du rapport Référentiel

A la racine du site, cliquer sur la rubrique "Administration du site" / "Notifications" ; le nouveau rapport est installé... cliquez sur ""Administration"/ "Administration du site" / "Plugins" / "Rapports" / "Gestion des rapports"
et vous devriez voir le nouveau rapport "Référentiel"

Utilisation du Module

1) Allez dans un cours (ou créez-le) et "Activez le mode édition" puis cliquez sur "Ajouter un bloc" et choisissez "Référentiel" (si elle n'apparaît pas c'est que vous avez omis de rendre ce bloc visible dans les paramètres de configuration des blocs...)

2) Vous pouvez alors utiliser le bloc Référentiel pour créer une nouvelle occurrence de référentiel ou télécharger un référentiel existant.

3) Puis revenez sur la gestion du cours et cliquez sur "Ajouter une activité", sélectionnez "Référentiel" et associez lui l'une des occurrences disponibles.
(On évitera d'associer différentes instances de référentiel à une même occurrence dans un même cours ; par contre il est tout à fait recommandé d'associer à une même occurrence des instances de référentiel déclarées dans des cours différents, cela afin de répartir la charge de l'évaluation entre les enseignants).

Pour éviter de ressaisir toute une occurrence de référentiel, vous pouvez télécharger des archives d'occurrences de qui se trouvent sur MoodleMoot 2009 dans le dossier "Sauvegardes de référentiels".: http://moodlemoot2009.insa-lyon.fr/mod/resource/view.php?id=849

Pour en savoir un peu plus consultez la documentation utilisateur qui est ici.

Mise à jour

Le module référentiel évolue rapidement en fonction des demandes des utilisateurs. Quand vous installez une mise à jour contentez-vous de la décomprimer dans le même dossier que votre version antérieure. Puis logez-vous comme admin, cliquez sur Notification... et lisez les messages.

Dans le dossier ./mod/referentiel/ consultez le fichier README.txt et dans le dossier ./mod/referentiel/documentation/ les dernières informations concernant l'évolution du module.

Ce qu'il faut éviter absolument lors d'une mise à jour serait de conserver la version ancienne -- même renommée -- dans le dossier ./mod/ car cela introduit de graves dysfonctionnements dans ce module et dans Moodle.


Marche à suivre pour désinstaller le module référentiel

1. Logez-vous avec le role admin

2. Supprimez le bloc Référentiel

Administration / Plugins / Activités / Gestion des activités

Sélectionnner Référentiel :: Supprimer

Supprimez le dossier VOTRE_DOSSIER_MOODLE/blocks/referentiel

3. Supprimez le module Référentiel

Administration / Plugins / Activités / Gestion des activités

Sélectionnner Référentiel :: Supprimer

Supprimez le dossier VOTRE_DOSSIER_MOODLE/mod/referentiel

4. Supprimez le rapport Référentiel

Administration / Plugins / Activités / Gestion des rapports

Sélectionnner Référentiel :: Supprimer

Supprimez le dossier VOTRE_DOSSIER_MOODLE/report/referentiel

5. Supprimer les dossiers

./moodledata/#course_id/moddata/referentiel de tous les cours où une instance a été déclarée

1. Unzip the archive ./mod/referentiel-mod.zip in moodle/mod/ directory

2. Unzip the archive ./blocks/referentiel-block.zip in moodle/mod/ directory

3. Unzip the archive ./report/referentiel-report.zip in moodle/mod/ directory

4. log on with admin role

5. install new module as usual (admin Notification)

6. Set module parameters
